# 🍔 MERN Food Delivery Web Application

A full-featured **Food Delivery Web Application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
The platform allows users to browse food items, place orders, and track their delivery in real time. It also includes a powerful **Admin Panel** for managing orders, products, and users.

---

## 🚀 Features

### 👤 User Features

- User Authentication (Signup / Login)
- Browse food items
- Add items to cart
- Secure checkout
- Place orders
- **Live order tracking**
- Order history
- Fully responsive design for all devices

### 🛠 Admin Panel

- Add / Edit / Delete food items
- Manage customer orders
- Update order status
- Manage users
- Dashboard overview of orders and products

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/food-delivery-app.git
```

2️⃣ Navigate to Project Folder
cd food-delivery-app
3️⃣ Install Dependencies

Backend:

cd backend
npm install
Frontend:

cd frontend
npm install

Admin Panel:

cd admin
npm install
4️⃣ Run the Application

Backend:

npm run server

Frontend:

npm start

Admin Panel:
npm start
